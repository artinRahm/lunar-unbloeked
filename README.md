# Unblocked Game Website

## Description

This is an unblocked game website created and maintained by me. The project is still in active development, with ongoing updates and improvements planned through 2027.

The goal is to build a reliable and expanding collection of browser-based games that are easy to access and play without restrictions.

## Overview

The site hosts a selection of fast, lightweight games that run directly in the browser. It is designed for simplicity, performance, and accessibility across different networks and devices.

## Included Games

* Ultrakill-inspired browser game
* Deltarune-inspired content
* Bad Time Simulator
* Sans Fight

## Features

* No downloads required
* Works in most modern browsers
* Lightweight and fast loading
* Simple interface
* Unblocked access

## Installation

```bash
git clone https://github.com/artinRahm/lunar-unbloeked
```

Open `index.html` in your browser to run the site.
