What is Lunar Unbloked? v1.0

Lunar Unbloked is a personal browser homepage built to feel like its own operating system. It runs entirely in a single .html file — no servers, no installs, no accounts. Drop it in a folder with your game files, open it in any browser, and you have an instant local arcade + dashboard with a built-in proxy.

How to Add Games

Place your .html game files in the same folder as lunar_unbloked.html. Then open the file in a text editor and find the const GAMES = [...] array near the bottom of the script. Add a new entry like this:


{ name: "My Game", file: "mygame.html", tag: "arcade", emoji: "🎯", bg: "linear-gradient(135deg,#f00,#00f)" }


Optionally add an image: "url" field to show a thumbnail instead of the emoji. Available tags: arcade, puzzle, action, strategy, word.

Pages & Navigation

VaultPlay has a tab system at the top and a dock at the bottom. Current pages:

⊕ Home / New Tab 🎮 Games 👥 Social 📄 Docs

Tabs open automatically when you navigate. You can add more tabs with the + button in the browser bar, and close them with the × on each tab.

Social / YouTube

The Social page lets you save YouTube channel links. Type a channel URL or name, hit ADD, and it gets saved to localStorage — so it persists across sessions. Click any saved channel card to open it in a new tab. Delete channels with the × button on each card.

Splash Texts

On the home page, the subtitle text below VAULTPLAY is a randomly picked splash message. Click it to cycle through all the messages. There are 15 built‑in splashes — add your own by editing the SPLASHES array in the script.

Tech Stack

Pure HTML + CSS + Vanilla JS. No frameworks, no build step, no dependencies. The particle background uses an embedded version of particles.js by Vincent Garreau. Fonts are loaded from Google Fonts (Space Mono + Syne). Everything else is hand-written and self-contained in one file.
